# Squid Game (무궁화 꽃이 피었습니다)

## 개요
드라마 오징어게임의 무궁화 꽃이 피었습니다 장면을 3d 게임으로 구현<br>

## SquidGame.html 실행 방법
open with live server 이용

주의 : 모델 로드시간이 길기 떄문에 반드시 버튼이 활성화 되고 시작할 것
